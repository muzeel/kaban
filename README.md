TaskFlow — Система управления проектами и задачами (Trello-подобное приложение)

Основные фичи (Features):

    1. Аутентификация и авторизация: Регистрация, вход, восстановление пароля. Роли: пользователь, администратор.

    2. Управление проектами: Создание, редактирование, удаление проектов. Приглашение других пользователей в проект.

    3. Доска задач (Kanban): Создание задач внутри проекта, перетаскивание их между колонками статусов ("К выполнению", "В процессе", "На проверке", "Готово").

    4. Детализация задачи: У задачи есть заголовок, описание, исполнитель, срок, приоритет, метки.

    5. Коммуникация: Комментарии к задачам, прикрепление файлов.

    6. Поиск и фильтрация: Фильтрация задач по исполнителю, статусу, меткам.

    7. Личный кабинет: Просмотр назначенных на пользователя задач.

    8. Административная панель: Управление пользователями и проектами
