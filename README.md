# TaskFlow - Система управления проектами и задачами

**TaskFlow** - это веб-приложение для управления проектами по методологии Kanban, созданное на Ruby on Rails. Приложение позволяет командам эффективно организовывать работу, распределять задачи и отслеживать прогресс.

## Ссылка на репозиторий
https://github.com/muzeel/kaban

## Основной функционал

### 1. Аутентификация и авторизация
- Регистрация новых пользователей
- Вход/выход из системы
- Восстановление пароля
- Разделение ролей: пользователь и администратор

### 2. Управление проектами
- Создание новых проектов
- Редактирование и удаление проектов
- Просмотр списка проектов
- Приглашение других пользователей в проект

### 3. Доска задач (Kanban)
- Создание задач внутри проектов
- 4 статуса задач: "К выполнению", "В процессе", "На проверке", "Готово"
- Drag-and-drop перемещение задач между колонками
- Изменение приоритета задач

### 4. Детализация задач
- Заголовок и подробное описание задачи
- Назначение исполнителя
- Установка сроков выполнения
- Выбор приоритета (Низкий, Средний, Высокий, Критический)
- Добавление меток (тегов)

### 5. Коммуникация
- Система комментариев к задачам
- Прикрепление файлов к комментариям
- Уведомления об изменениях (в будущих версиях)

### 6. Поиск и фильтрация
- Фильтрация задач по статусу
- Фильтрация по исполнителю
- Поиск по названию и описанию
- Фильтрация по меткам

### 7. Личный кабинет
- Просмотр задач, назначенных на пользователя
- Статистика выполненных задач
- Настройки профиля

### 8. Административная панель
- Управление пользователями
- Просмотр статистики по проектам
- Модерация контента

## Технологический стек

### Бэкенд
- Ruby 3.x
- Ruby on Rails 7.x
- PostgreSQL 14+

### Фронтенд
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Stimulus.js для интерактивности
- Hotwire (Turbo) для SPA-эффекта

### Аутентификация и безопасность
- Devise для аутентификации
- Pundit для авторизации
- BCrypt для хеширования паролей

### Дополнительные гемы
- rails_admin для админ-панели
- carrierwave для загрузки файлов
- kaminari для пагинации
- rspec для тестирования
